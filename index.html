Program Whitening Treatment

ให้ผิวของคุณกลับมากระจ่างใส ลดจุดด่างดำด้วยการผลัดเซลล์
ผิวอย่างอ่อนโยน เหมาะสำหรับผู้ที่มีปัญหาผิวหมองคล้ำรอยดำ
จากสิวช่วยฟื้นฟูผิวให้ชุ่มชื้นดูสุขภาพดี

ด้วยผลิตภัณฑ์นำเข้าจาก
สหรัฐอเมริกาและยุโรป

ฟื้นฟูและเพิ่มความชุ่มชื้นให้ผิว

ไฮยาลูรอนิค ลดริ้วรอยและเพิ่มความชุ่มชื้น

Endocare

แบรนด์สเปนใช้สารสกัดหอยทาก
กระตุ้นคอลลาเจนและฟื้นฟูผิว

Program Nourishing Treatment

เผยผิวใสเปล่งปลั่งเหมาะสำหรับผู้มีปัญหาผิวหมองคล้ำ
รอยสิว ผิวแห้ง ขาดความชุ่มชื้น

  • ผลัดผิวด้วย Gommage
  • บำรุงผิวด้วย Endocare
  • นวดผ่อนคลาย 16 ขั้นตอน
  • มาส์กกระจ่างใสด้วยกรดผลไม้
  • บำบัดเย็นด้วย Cryotherapy
  • มาส์กเพิ่มความชุ่มชื้นจาก Shea Butter
  • มาส์กวิตามินจาก Sea mud และสาหร่าย

Program Lymphatic Treatment

ผสานเทคนิคนวดกัวซ่าจากตะวันออกและตะวันตก
กระตุ้นการไหลเวียนโลหิตและเพิ่มการสร้างคอลลาเจน

  • บำรุงผิวด้วย Q-Skinscience
  • กัวซาเย็นด้วยหยกขาวเพื่อยกกระชับ
  • มาส์กเพิ่มความใสด้วย Zinc และ Sulfur
  • มาส์กชุ่มชื้นจาก seaweed และ chamomille
  • กัวซาร้อนเปิดผิวและกระตุ้นและระบายน้ำเหลือง

Program Hydrating Treatment

เพิ่มความชุ่มชื้น ลดการอักเสบ เหมาะสำหรับผิวแห้ง
อักเสบ หรือไหม้แดด

  • บำบัดเย็นด้วย Cryotherapy
  • นวดผ่อนคลายใบหน้า 11 ขั้นตอน
  • ผลัดผิวด้วยกรดผลไม้อย่างอ่อนโยน
  • กัวซาร้อนเปิดผิวและกระตุ้นและระบายน้ำเหลือง
  • มาส์กชุ่มชื้นจาก Honey, Almond, และ Seaweed
